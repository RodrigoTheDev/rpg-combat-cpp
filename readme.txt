================ AVISO ================

Devido à alguns problemas com o repositório antigo, ele acabou sendo apagado, esse novo repositório será a versão atualizada

=======================================

Esse jogo é um projeto de um sistema de combate rpg, onde o jogador e o monstro tem a vez de atacar.

O código é dividido em arquivos hpp, fiz isso para deixar o código não só mais organizado, como também modular, de modo que não haja dificuldades desnecessárias na hora de adicionar novas features

as funções estão devidamente comentadas, mas correções podem surgir futuramente
